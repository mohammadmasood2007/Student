
# 🎓 Student Goal Tracker

A Python + Streamlit application that helps students manage learning goals, track progress, and visualize study performance in a simple and structured way.

## 🚀 Features
- Add, update, search, and delete goals  
- Track progress for each goal  
- Interactive Streamlit dashboard  
- Charts and analytics  
- Overdue goal alerts  
- Study suggestions based on progress  

## 📈 Analytics
- Total goals  
- Completed goals  
- Pending goals  
- Overdue goals  
- Average progress  

## 🛠 Tech Stack
- Python  
- Streamlit  
- Pandas  
- Plotly  
- JSON

```bash

cd Student
pip install -r requirements.txt
python AI_project.py
streamlit run dashboard.py
````
